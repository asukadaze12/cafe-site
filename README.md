# カフェサイト

## 概要
架空の猫カフェのWebサイトです。
落ち着いた雰囲気と見やすさを意識して制作しました。
静かな時間を過ごせる空間、癒しのカフェをイメージしました。

## URL
https://asukadaze12.github.io/cafe-site/

## 使用技術
- HTML
- CSS

## 制作について
Web制作の学習を目的として、ChatGPTを活用しながらサイト全体を制作しました。
分からない点を確認しつつ、自分で手を動かして形にしています。

## 工夫した点
- 写真を大きく使って雰囲気が伝わるようにした
- 文字が読みやすくなるよう配色を調整した
